This is a plugin for Skynet integration with WooCommerce

== Description ==
Skynet Malaysia is the official extension for WooCommerce on WordPress for Skynet Malaysia.

FEATURES
- Easily create Skynet Airwaybill sticker for your WooCommerce orders directly from your WooCommerce order page.

== Installation ==
= Requirements =

- WooCommerce 3.0.0 or later
- SkyPostpaid account token

= Installation instructions =

1.a Upload the plugin files to the `/wp-content/plugins/` directory
OR
1.b Go to Plugins menu > add new plugin and search for `Skynet Malaysia` in your Wordpress Admin panel.

2. Activate the plugin through the \'Plugins\' menu in WordPress

3. Fill up your company information and API token in WooCommerce > Settings > Shipping > Skynet. if you do not have an API token or do not have a Skynet Account, you may register an account and generate a token at https://skynet.com.my/

== Frequently Asked Questions ==
= Can I use this plugin without setting up WooCommerce? =
No. This plugin does not work without WooCommerce installed and set up.

= Do I have to have a SkyPostpaid account to use this plugin? =
Yes. this plugin requires a unique token that will be binded to your SkyPostpaid account.

= Where can i create a SkyPostpaid account? =
You may create an account at https://www.skynet.com.my/apply_account and then requesting a SkyPostpaid Login.
